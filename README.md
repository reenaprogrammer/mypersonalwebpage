## 🛠️ Tech Stack & Features
* **HTML5 & CSS3:** Semantic structure with a fully responsive layout using Flexbox and CSS Variables.
* **JavaScript (ES6+):** * Dynamic content generation for the "Book Shelf."
    * State management using `localStorage` to remember theme preferences.
    * Conditional logic to ensure smooth navigation across multiple pages.
* **Dark Mode:** A custom-built theme switcher that persists across the Home, Story, and Contact pages.
* **Interactive UI:** CSS-animated flipping cards for project/book details.

## 📂 Project Structure
* `index.html`: The landing page featuring my interactive project/book shelf.
* `story-detail.html`: A deep dive into my career transition and personal motto.
* `contact.html`: A professional contact portal with social media integration.
* `style.css`: The "brain" of the design, handling layouts and themes.
* `script.js`: The engine handling all interactive logic.
